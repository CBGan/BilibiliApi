# 好耶，是API

> BiliBili的部分API封装库

写这个库是因为想更方便的使用B站的Web API

对API的接口进行了封装

并且API返回值也封装为了抽象类

因为这个库是从[SuiseiBot](https://github.com/CBGan/SuiseiBot)项目中单独分离出来的库，所以这个库的更新进度也和机器人的使用有关联
